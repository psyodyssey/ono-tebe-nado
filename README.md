# Оно тебе надо

Учебный проект Яндекс Практикума — лендинг аукциона необычных вещей. Статическая страница, свёрстанная по макету с использованием семантического HTML и методологии БЭМ.

## Стек

- HTML5 (семантическая разметка)
- CSS3 (методология БЭМ)

## Структура проекта

```
├── index.html
├── styles/
│   ├── global.css   — reset/normalize
│   └── style.css    — основные стили
├── images/          — изображения и иконки
└── fonts/           — шрифты проекта
```

## Запуск

Откройте `index.html` в браузере — сборщик не требуется.

## Статус

Проект завершён. Вёрстка всех секций выполнена по макету, шрифты и изображения подключены.
